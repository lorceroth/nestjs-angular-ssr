This repository is a sandbox to play around with Server-Side Rendering using NestJS and Angular.

## Requirements
- Node >= 8.9
- NPM >= 5.5.1

## Installation

```bash
$ git clone https://github.com/lorceroth/nestjs-angular-ssr.git
$ yarn # or npm install
```

## Running the app

```bash
# development
$ yarn serve

# production
$ yarn build
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn e2e
```
